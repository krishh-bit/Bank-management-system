# Bank-management-system

A simple console-based Bank Management System built using Java and Object-Oriented Programming (OOP) concepts. This project simulates basic banking operations and provides a user-friendly menu-driven interface for managing customer accounts.

## Features

* Create New Account
* View All Accounts
* Search Account by Account Number
* Deposit Money
* Withdraw Money
* Check Account Details
* Menu-Driven Interface

## Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)
* Scanner Class

## Concepts Implemented

* Classes and Objects
* Encapsulation
* Methods
* Collections Framework (ArrayList)
* User Input Handling

## Future Improvements

* Account Deletion
* Transaction History
* Interest Calculation
* File Handling for Data Storage

This project was developed as part of my Java learning journey and to strengthen my understanding of Object-Oriented Programming concepts.
